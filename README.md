# playdl
A Linux program to download YouTube playlists as both videos and high quality audio.
### Dependencies
```
python
atomicparsley
ffmpeg
youtube-dl
zenity
```
### Download & Usage
```shell
git clone https://github.com/aliraslan/playdl
cd playdl
bash playdl.sh
```
### Installation
